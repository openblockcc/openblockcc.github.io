## 编辑 Wiki

- 开发环境

    python3.x

- 安装

    ```bash
    pip install mkdocs-material
    ```

- 预览页面

    ```bash
    mkdocs serve
    ```

    运行上述指令启动 mkdocs 本地服务器并使用浏览器访问 http://127.0.0.1:8000 即可预览部署网站后的效果。

- 发布

    工程已配置 Github Action， 提交代码后会构建站点内容。
